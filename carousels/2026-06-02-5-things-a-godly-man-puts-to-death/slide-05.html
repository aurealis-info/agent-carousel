Thing 4

The laziness you call a season

There’s real rest — Sabbath, recovery, the land lying fallow. God built that in. That’s not what this is.

This is six months of avoiding the hard thing and framing it as discernment. Proverbs visits the sluggard nine times. None of it is gentle.

  • Name what you haven’t started. Don’t dress it up.
  • Set a start time today — not a plan, not a goal. A time.

Sloth never wears its own name.
